# Mahabharat Scholar - Critical Edition Reference

A comprehensive web application for scholarly study of the Mahabharata epic, based on the BORI Critical Edition with support for multiple languages, editions, and advanced research tools.

## Features

### 🔍 Advanced Search Capabilities
- **Verse Search**: Search through thousands of verses using keywords, phrases, or concepts
- **Character Search**: Find all verses related to specific characters
- **Episode Search**: Explore verses by major episodes and events
- **Theme Search**: Discover verses by thematic content (dharma, karma, war, etc.)
- **Advanced Filtering**: Filter by parva, character, verse range, and edition

### 🌐 Multi-Language Support
- **English**: Modern scholarly translations
- **Hindi**: Devanagari script with Roman transliteration
- **Sanskrit**: Original text in Devanagari with Roman transliteration
- **Language Toggle**: Switch between languages seamlessly

### 📚 Scholarly Features
- **BORI Critical Edition**: Primary source based on the most authoritative scholarly version
- **Multiple Editions**: Compare BORI, Vulgate, Bengali, Southern recensions
- **Credible Sources**: All content fact-checked against peer-reviewed scholarship
- **Citation System**: Proper scholarly citations and references
- **Edition Comparison**: Side-by-side comparison of different textual traditions

### 👥 Character & Relationship Mapping
- **Family Trees**: Interactive D3.js visualizations of character relationships
- **Character Profiles**: Detailed information about each character
- **Relationship Mapping**: Understand complex family connections
- **Character Categories**: Pandavas, Kauravas, Gods, Sages, Warriors, Women

### 📖 Episode & Theme Organization
- **Episode Categorization**: Organized by major events and parvas
- **Theme Classification**: Categorized by philosophical and narrative themes
- **Cross-References**: Links between related verses, characters, and themes

## Usage

### Basic Search
1. Enter search terms in the main search box
2. Select search type (verses, characters, episodes, themes)
3. Use advanced filters to narrow results
4. Click on results to view detailed information

### Family Trees
1. Navigate to Family Trees section
2. Select a family tree (Kuru Dynasty, Pandu Lineage, etc.)
3. Interact with the visualization to explore relationships
4. Click on characters for detailed information

### Edition Comparison
1. Go to Edition Comparison section
2. Select 2 or more editions to compare
3. Enter a verse reference
4. View side-by-side comparison of different textual traditions

### Language Support
1. Use the language toggle button in the header
2. Switch between English, Hindi, and Sanskrit
3. View original text with transliterations
